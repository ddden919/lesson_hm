# js 数据类型


- js有多少种数据类型？
  ** 内存分配机制 **
  ** 拷贝，  引用 **


    依据变量内存的分配机制，简单数据类型存在栈中，复杂数据类型存在堆中地址存在栈里，所以js有两种数据类型
    简单数据类型是拷贝，复杂数据类型是引用
  - 高手会讲八种
  - 高高手会讲七种
  - 简单数据类型 Primitive
    一眼就能看出其值

    函数，数组等有很多特性无法表达
    numeric(number + bigint) string boolean undefined null symbol 
  - 复杂数据类型 Object

   - null
    null 表示一个空值或不存在的对象，是一个可以赋给变量的特殊值。
    显示回收内存

   - undefined 未定义(类型)

   - symbol 唯一值
     - 以函数的形式创建，简单值
     - 打个标签
     - 哪怕标签一样，他的值也不相等

   - bigint 大整数

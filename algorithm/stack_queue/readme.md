FIFO Queue

- typeof
  变量存储的二进制 前三位 
  000 :表示对象
  001 : 字符串
  010 : 数组
  011 : 函数
  100 : 整数
  101 : 浮点数
  110 : 指针
  111 : 其他

- Object.prototype.toString.call()

- substring slice
  - substring 截取字符串
  - slice 截取数组 第二个参数endIndex，不包括，支持负数，反向取更灵活
  - substring 第二个参数endIndex，不包括，不支持负数，只能正向取
  - 第二个参数 不一样
